# CommitGPT

Generate your git commig message using AI. 

## Setup

- The tool is based on openAI chat gpt. You need to have their api key set to use the tool.

## Usage

- Set the `OPENAI_API_KEY` environment variable.
- `brew install commit-gpt`
- In any repository folder,  you can use command `commit-gpt generate` and it will generate messages based on the file changes.
- You can interact with the tool to enhance the message. for example you can ask: `please generate more verbose message`. It will generate more verbose message.
